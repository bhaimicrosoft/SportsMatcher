# Create Ionic App (Angular) to help a user locate nearby playgrounds for various sports like PickleBall / Tennis / Badminton etc.

- It should be designed using the latest Ionic framework using Angular.
- The name will be SportsMatcher
- It should use user's current location to locate the nearby playgrounds
- The app should be modern with attractive UI
- It should support authentication like signup and signin using email + password as well as using Biometric such as Fingerprint / FaceId
- It will be published to both Google PlayStore and Apple AppStore
- For designing, use TailwindCSS v4 classes and themes.
- For any questions, ask me
